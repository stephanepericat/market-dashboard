# Market Dashboard

A demo market monitoring terminal built with Vue.js and Electron.

## Run the App Locally

```shell
git clone https://github.com/stephanepericat/market-dashboard.git

cd market-dashboard/

yarn
yarn serve:electron
```

## Run Tests

```shell
yarn test
```

## Build

```shell
yarn build:electron
```
